# HR 智慧诊断平台 · HR Analytics Dashboard

> 纯浏览器端 HR 数据分析工具，上传 Excel 即可生成可视化图表与诊断报告。  
> A browser-only HR analytics tool. Upload your Excel file to generate charts and diagnostic reports instantly.

**[🚀 在线体验 Live Demo](https://zgy0606.github.io/hr-analytics-dashboard/)**

---

## ✨ 功能特性

### 招聘数据分析
- 📊 招聘漏斗（投递 → 面试 → 录用 → 到岗 → 留任）
- 📈 渠道转化效率对比（各渠道面试通过率 & 录用率）
- 🏢 部门招聘分布、性别 / 年龄 / 学历结构
- 🔍 招聘数据质量诊断（字段填充率 & 异常检测）

### 员工数据分析
- 👥 人力规划：部门分布、司龄、年龄、性别结构
- 🎯 绩效管理：分布直方图、等级构成、部门对比、预警名单
- 💰 薪酬福利：成本构成、部门薪酬对比、薪资区间分布
- 🔬 诊断报告：
  - **组织风险雷达**（5维健康扫描）
  - **组织健康得分**（综合评级 A/B/C/D）
  - **员工群体画像**（绩效 × 司龄 四象限矩阵）
  - **数据质量诊断**（缺失值 / 异常年龄 / 重复员工 / 不合理薪资）
  - **组织健康分析报告**（数据驱动，可一键复制）

## 🔒 隐私保障

**所有数据均在本地浏览器处理，不上传任何服务器。**  
您的 HR 数据绝不会离开您的电脑。

All processing happens entirely in your browser — no data is ever sent to any server.

---

## 🚀 使用方法

1. 打开网页（或直接用浏览器打开 `index.html`）
2. 选择模块：**招聘数据分析** 或 **员工数据分析**
3. 上传 Excel 文件，或点击「使用演示数据」快速体验
4. 系统自动识别列名并生成图表，缺失列自动跳过

### Excel 列名说明

**招聘数据**所需列名：
```
姓名、性别、年龄、学历、应聘部门、应聘岗位、招聘渠道
投递日期、是否面试、面试结果、是否录用、实际到岗日期、是否在职
```

**员工数据**所需列名：
```
员工编号、姓名、性别、年龄、学历、部门、岗位
入职日期、在职状态、离职日期、绩效评分、绩效等级
基本工资、绩效奖金、公司社保、公司公积金、其他福利
```

> 💡 无需严格对应列名顺序，系统自动识别；列名缺失时对应图表自动跳过。

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| 纯 HTML / CSS / JavaScript | 无框架，零依赖构建 |
| [Chart.js 4.4](https://www.chartjs.org/) | 图表渲染 |
| [SheetJS (xlsx)](https://sheetjs.com/) | Excel 文件解析 |

**单文件设计**：整个应用就是一个 `index.html`，下载后可直接在浏览器中离线使用。
